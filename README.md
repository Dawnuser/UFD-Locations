# UFD Locations

A family location directory. Tap an NFC tag to open a clean list of important places — house addresses, hypermarkets, hospitals — and navigate to any of them with Google Maps.

Built for the family door. Tap and go.

## Usage

1. Open `https://Dawnuser.github.io/UFD-Locations`
2. Tap a location card
3. Google Maps opens with directions to that place

## Files

- `index.html` — Flat list with emoji markers (default)
- `categorized.html` — Grouped by category

## Adding Locations

Edit the HTML files and push to GitHub. The NFC tag URL stays the same.

## Tech

Single-file HTML, no frameworks, no build step.
